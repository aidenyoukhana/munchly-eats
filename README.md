## Features

- **Browse Restaurants** - Discover nearby restaurants with ratings, delivery times, and fees
- **Menu Browsing** - View detailed menus with customization options
- **Smart Cart** - Add items with special instructions and track your order total
- **Order Tracking** - Real-time delivery tracking with driver location on map
- **User Profiles** - Save addresses and payment methods for quick checkout

## Tech Stack

- SwiftUI
- SwiftData
- MapKit
- Async/Await

## Requirements

- iOS 17.0+
- Xcode 15.0+

## Getting Started

1. Clone the repository
2. Open `MunchlyEats.xcodeproj` in Xcode
3. Build and run on simulator or device
